This is a view that draws radar chart diagram.

## Download

Grab via Gradle:

```groovy
compile 'com.dzaitsev.android.radarchartview:radarchartview:0.1.0'
```

Maven:

```xml
<dependency>
    <groupId>com.dzaitsev.android.radarchartview</groupId>
    <artifactId>radarchartview</artifactId>
    <version>0.11.0</version>
</dependency>
```
